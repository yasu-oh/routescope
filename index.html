RouteScope

Cisco IOS / IOS-XE / NX-OS の基本的な IPv4 / IPv6 経路表を構造化し、VRF・prefix・protocol・AD/metric・next-hop/interface 単位で比較します。

入力データはブラウザ内だけで処理されます。サーバ送信、外部API、analytics、CDN、自動保存は使いません。

Input

Before / After の show ip route または show ipv6 route 出力を貼り付けてください。

Summary

Added / Removed / Changed / Unchanged を集計します。
